## Variable(변수)
- 메모리에 이름을 붙여넣고 값을 저장하는 것을 변수라고 한다. 
- 파이썬의 변수는 별도의 타입을 지정하지 않는다.
- 타입은 선언에 의해 결정된다.
- 변수의 이름은 스페이스를 포함하지 않음
- 변수의 이름은 특수문자를 제외한 알파벳 문자와 숫자, 그리고 언더스코어만(_)만 포함할 수 있음

- 파이썬 내부에서 사용하는 키워드를 사용할 수 없음 
	- and, del, from, not, while, as, elif, or, class, global, with
- 변수의 이름은 대문자와 소문자를 구분함

if문
- 조건문의 참, 거짓을 판별하여 명령의 실행 여부를 결정하는 제어문이다.
- 순서도를 작성하기에 좋다. 어디에서 설명을 해야 할 때, 내가 무엇을 만들었는데, 이러한 로직을 사용했다. 로직을 짰을 때 이러한 식으로 구성을 했다. 
- 들여쓰기에 따라서 블록의 수준이 결정된다. 그러므로 들여쓰기 깊이가 같다면 여러명령을 실행시킬 수 있다.

```python
# 조건문과 비교연산자를 활용한 예제
a = 5
if a >= 5:
    print('a는 5보다 크거나 같다.')
if a != 6:
    print('a는 6이 아니다!')

# 조건문과 논리연산자를 활용한 예제
a = 3
b = 5
if a == 3 and b == 5: 
    print('a = 3, b = 5') # a=3,b=5 True
if a == 5 or b == 5:
    print('a=5, b=5 or')
```

## Loop_Structure
반복문은 프로그래밍 언어가 가장 프로그래밍 언어답도록 만드는 가장 기본적인 규칙이라는 생각이 든다. 사람은 반복하는 일에 능하지 않지만 컴퓨터는 몇번의 반복이라도 정확하게 수행할 수 있다. 이 지점 때문에 사람이 하기 어렵고 복잡한 일을 프로그래밍를 통해서 컴퓨터에게 수행하게 할 수 있다. 

- while loop, 횟수가 정해지지 않은 반복
- for loop, 횟수가 정해진 반복
- 제어변수의 활용
- break
- continue



#### 제어변수
반복문은 특정 변수값을 기준으로 반복하는데 이 때 기준이 되는 변수가 제어 변수(Control Variable)이다. 이 때 제어변수가 중요한 이유는 횟수가 정해진 반복, 횟수가 정해지지 않은 반복을 수행할 때 반복문의 제어가 가능해야 하는데, 이 역할을 제어변수가 수행하기 때문이다.

**while문은 반복의 횟수가 정해지지 않았을 때 사용한다**  
```python
number = 0
while number <= 3:
  print('hi')
  number += 1
--- 
hi
hi
hi
```

**for문은 반복의 횟수가 정해졌을 때 사용하다**  
```python
# for 문을 사용해 구구단 출력
for dan in range(2, 10):
    print(dan, '단')
    for hang in range(1, 10):
        print(dan, 'x', hang, '=', dan*hang)
    print()
```
**for문을 더 잘 활용하기 위해서 range 함수를 이용할 수 있다.**
- 시작값을 첫요소로 만들고 끝값 직전까지 값을 생성  

```python
range(1, 5)  # 1,2,3,4
# 아래처럼 세번째 인자로 값을 전달하여 증가폭을 조정할 수 있다.
range(1,10,2) # 1,3,5,7,9 


```

**break**  
실행되는 반복문을 벗어나야 할 때 사용하는데, 반복의 한 주기만을 중지하는 게 아니라 반복전체를 중지한다.

```python
num = 0
while num < 10:
    if num == 5:
        print('\n5가 되면 반복문의 실행을 벗어납니다.')
        break
    print(num, end='')
    num += 1


```


**continue**  
지속되는 반복문의 실행 안에서 '반복 전체를 중지하는 것이 아닌 한 주기의 실행'만을 중지하고 다음 문장으로 넘어가야 할 때 사용하는 문법이다. 

```python
for num in range(1, 11):
    if num % 2 == 0:
        # print(num, end='')
        continue
    else:
        print(num, end=' ')
```

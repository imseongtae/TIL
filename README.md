#### 내용을 더 잘 정리하기 위한 규칙:card_file_box:

- 프로그래밍 언어/프레임워크/라이브러리/프로그램/강연 등의 이름으로 폴더를 만든다.
- 학습한 내용, 기억하고 싶은 내용을 정리한다.
- 학습한 내용이 작은 부분일지라도 넘기지 않고 정리한다.
- 직접 학습한 내용만 정리한다.
- 내용 채우기만을 위한 복붙은 하지 않는다

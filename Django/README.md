## config setting

`config` 뒤에 `.`을 찍지 않으면 config 폴더 아래에 config 폴더가 생기므로  `config .`로 프로젝트를 생성할 수 있도록 주의한다

```
django-admin startproject config  # X
django-admin startproject config . # O
```

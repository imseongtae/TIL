## 단축키

- `ctrl`을 누르고 마우스로 할 수 있는 일이 많다!
- `alt`를 통해 간격을 잡을 수 있다. 

| 단축키        | 기능               |
| ------------- | ------------------ |
| 잠그기        | `ctrl + shift + L` |
| 그룹화        | `ctrl + G`         |
| 이름 변경     | `ctrl + R`         |
| 컴포넌트 생성 | `ctrl + alt + k`   |
| Rectangle     | `R`                |
| Arrow         | `shift + L`        |
|               |                    |
|               |                    |
